# Unit Converter Android App (Java)
A modern Android Unit Converter application built using Java and Android Studio. This app allows users to quickly convert values between different units such as Length, Weight, Temperature, Area, Volume, and more through a simple and user-friendly interface.

---

## 🚀 Features

✅ Length Conversion  
✅ Weight Conversion  
✅ Temperature Conversion  
✅ Area Conversion  
✅ Volume Conversion  
✅ Material Design UI  
✅ Dark Mode Support  
✅ Conversion History  
✅ Fast and Accurate Calculations  
✅ User-Friendly Interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/mangurajubhukya-cmyk/Unit-Converter-.git
```

### Open in Android Studio

1. Open Android Studio
2. Select **Open Existing Project**
3. Choose the cloned project folder
4. Sync Gradle
5. Run the application

This is the Unit Converter Android application created for the Oasis Infobyte internship project. It supports conversions for Length, Mass, Temperature, Volume, Speed, Data Size, Area, Time, and Pressure. It also supports custom linear units, swap, history, and copy-result features.

How to run:
1. Open this folder in Android Studio.
2. Let Gradle sync (Android Studio will download required SDK components).
3. Run on an emulator or connected device (minSdkVersion 21).

Features:
- Length, Mass, Temperature, Volume, Speed, Data Size, Area, Time, Pressure conversions
- Add custom linear units (persisted locally)
- Swap from/to units
- Conversion history (persisted locally)
- Tap result to copy
